    关于书香中国爬虫制作

    Posted on 2020-09-11 Edited on 2020-12-12 In Python

    环境

    • 语言：Python3
    • IDE：PyCharm
    • selenium
    • pyautogui

    遇到的问题

    本来只是想单纯的用selenium这个自动化测试的库来对网页进行pdf的爬取，但是由于书香中国中的pdf都是以iframe的标签来展示的，无法具体捕捉到里面的内容，所以我使用selenium+pyautogui的方案来模拟手动下载图片的过程

    代码

    import pyautogui
    from selenium import webdriver
    from selenium.webdriver import ActionChains
    import time

    所要爬取的页面地址

    uri = ‘’
    browser = webdriver.Chrome()
    browser.get(‘http://www.chineseall.cn/org/index.action')

    登入书香中国的账号！这一点很重要！！

    这里将272改变成你需要的pdf的页数

    for i in range(272):
    browser.get(uri + str(i+1))
    pic = browser.find_element_by_id(‘bookContent’)
    time.sleep(5)
    action = ActionChains(browser).move_to_element(pic)
    action.context_click(pic).perform()
    time.sleep(1)
    pyautogui.press(‘f12’)
    pyautogui.press(‘enter’)
    time.sleep(3)
    pyautogui.press(‘enter’)
    time.sleep(3)
    browser.close()
