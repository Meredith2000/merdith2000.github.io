搭建owncloud

Posted on 2020-07-20 Edited on 2020-12-12 In Linux

前言

我的计划是搭建一个个人云盘，一开始我选择的是nextcloud，但是在我的一波操作之后，云盘的确是能顺利的安装到服务器上，但是问题在于，当我再次登录的时候，它总是显示密码错误，无限卡死在登录界面，我在网上查阅了很多资料，但是都是以失败告终！接着我有想尝试seafile，但是，我看了一下安装说明，似乎有点麻烦，于是我就选择了owncloud（听说这两个原来是一家）。

操作环境

  • CentOS 版本： 7.7
  • Apache 版本：2.4.6
  • PHP 版本： 7.4.7
  • Mysql 版本：8.0.20
  • owncloud版本：10.4.1

具体步骤

  • 输入一下命令（用yum安装，很简单）

rpm –import https://download.owncloud.org/download/repositories/production/CentOS\_8/repodata/repomd.xml.key
dnf config-manager –add-repo http://download.owncloud.org/download/repositories/production/CentOS\_8/ce:stable.repo
dnf clean all
dnf install owncloud-files; dnf upgrade owncloud-files

  • 提示没有dnf这个指令，就执行一下命令（如果一切正常请忽略）

yum install dnf
yum install config-manager

  • 由于这个yum源在国外，如果出现下载很慢或者下载rpm失败的话，可以先用一下地址下载好了rpm安装包，然后用ftp上传到Apache中去。
    https://download.owncloud.org/download/repositories/production/CentOS_8/
