    docker 安装

    Posted on 2020-07-20 Edited on 2020-12-12 In docker

    前言

    最近学习了docker这种容器虚拟化的技术，觉得这种技术真的很好，特别是适合我这种配置老是出问题的小白。但是今天我在安装docker的时候踩了大坑，由于用来官方的yum源，速度慢到怀疑人生。

    安装环境

    • CentOS 版本： 7.7
    • docker 版本：19.03.12

    安装步骤（不建议学习，但是可以参考一下）

    • 输入一下命令（在下载docker的时候会慢到怀疑人生）

    # 先移除老版本的docker（没有安装过的，请忽略）
    yum remove docker \
    docker-client \
    docker-client-latest \
    docker-common \
    docker-latest \
    docker-latest-logrotate \
    docker-logrotate \
    docker-engine

    初始化仓库

    yum install -y yum-utils
    yum-config-manager \
    –add-repo \
    https://download.docker.com/linux/centos/docker-ce.repo

    安装docker（极其慢，但是可行）

    yum install docker-ce docker-ce-cli containerd.io

    其他

    配置国内的阿里云镜像加速

    • 打开 /etc/docker/daemon.json，添加一下代码。（一定要先运行docker，再配置，如果不存在这个文件，就自己创建一个）

    {
    “registry-mirrors”: [“阿里云提供的地址”]
    }

    阿里云上的docker源

    # 添加yum源
    yum update
    yum install epel-release -y
    yum clean all

    安装并启用docker

    yum install docker-io -y
    systemctl start docker
