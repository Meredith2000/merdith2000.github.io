Linux Shell编程总结

Posted on 2020-07-18 Edited on 2020-12-13 In Linux

这是我对Linux脚本编写的一点浅薄的认识，如果有错误，请多多指正

关于引号的问题

  • 单引号 纯粹的普通字符
  • 双引号 一般当做普通字符来使用，但是”$” , “`“ , “\“ 仍然被看做是特殊字符
  • 倒引号 被shell解释为可执行的命令

shell中的各种运算符号

  • “$” 是取值运算，是取得变量的值
  • “${ }” 这是将花括号里面的东西看做一个整体

关于位置参数

  • $# 表示输入了几个参数
  • $?可以知道上一次程序执行的情况
  • 用set命令可以在shell中设定参数

set 1 2 3
echo $1 $2 $3

  • 每执行一次shift，命令行上面的参数都会向右移动一位（$0除外）

for 语法

do
…
done

for i in 文件正则表达式
do
…
done

for i in $*
do
…
done

# 与上面一种格式完全等价
for i
do
…
done

for ((i=0;i<len;i++))
do
…
done

etc

  • shell中的变量一律当做字符串
  • 在变量赋值的时候要是内容中有空格，一定要用引号引起来
  • [*]是表示将一个数组组合成一个字符串，[@]表示将数组分开输出
  • 在for循环中使用数组最好把in后面的数组用双引号引起来
  • 字符串里面不要放~，shell无法解析
