使用Vue来搭建南京林业大学马院自动化考试网站

不多废话，先上网站！

网站：https://service.zkwzkw.top

开源地址：https://github.com/Meredith2000/OnlineTestForNJFU

前言

这段时间在家里无聊，心血来潮学习了一下Vue，但是由于本人非常菜，所以各种查资料看视频，终于花了两天时间，模仿我们学校的马院的自动化考试网站，写了Vue的版本来帮助我复习（实际我发现有这时间写网站，不如去多背两个题😂），既然网站都写出来了，那我就不妨分享不来吧，也能帮助一些正在学习Vue的小伙伴！但是，我现在这个版本还非常的初级，仅仅只有单选题和判断对错的功能，后续会加上更多的题型和更多有趣的功能！

关于Vue

为什么我会选择Vue？

  • 没有为什么，仅仅只是因为觉得Vue牛X

在开发中Vue带给我最大的便捷

  • Vue中的响应式的特性给我免去了很多不必要的麻烦
  • Vue中组件化的设计让我再开发中极大的缩小了代码量，提高了代码的复用率
  • Vue中vue-router让我快捷的实现了前端路由，完成了所谓的SinglePage Web Application(SPA)
  • 同时吹爆vue cli，简直太好用了，还有Vue UI对我这种小白来说的真的是非常友好。

开发环境

  • WebStrom 版本：2020.1.1
  • Vue 版本：2.6.11
  • npm 版本：6.14.4
  • vue cli 版本：4.4.6
  • vue-router 版本：3.2.0
  • axios 版本：0.19.2

基本搭建过程

这里我不打算交代整个搭建步骤，因为网上搭建的步骤很多，我就不“重复造轮子”啦，我下面只说一个重点的部分

整个项目我都是通过vue cli来搭建，所以常用的命令就以下四条：

vue create online_test_for_njfu # 直接使用脚手架来初始化项目，美滋滋
vue run serve # 预览整个项目
vue run build # 编译整个项目
vue ui # vue中吹爆的一个功能，它将整个项目的配置的全部都以可视化的界面呈现出来，简直就是小白的福音

在项目搭建完成之后当然是构建组建啦，我的网站分为四个组件，分别是：

  • Header（这是整个网页的导航来，依靠它来跳转页面）
  • SideBar（本来想要通过它来实现不同题型的切换的，但是我现在的网站只有单选题，所以只好展示将其作为公告栏）
  • SingleSelectForXXX（这个就是网站最重要的部分啦，这个就是题目的显示与答题区）
  • router（实现前端路由，非常的方便和好用）

下面就让我来具体说说这四个组成部分吧！

Header

这一部分的功能很简单，就是放一些自己的logo还有slot标签来在外部能够对Header进行自定义，达到一个能够复用的效果。

在设计Header时候，遇到最大的问题就是slot标签中的内容无法在Header组件中改变样式，但是没关系，我在App中css顺利的改变了。

SideBar

这一部分的功能目前来说只是一个呈现公告的地方，所以他的制作也是非常简单的，主要就是与后面的组件（SingleSelectForXXX）组成一个flex-box，以此实现在一行显示的效果。

SingleSelectForXXX

这个就是整个网站的核心组件了！

这个组件我是将发送请求的功能和这个组件的框架全部耦合在了一块，因为我觉着这样能使整个模块的功能更加清晰，我选择axios来发送get请求，在我的后台web服务器中有多个servlet能够返回每一题的全部内容。

同时在显示颜色方面也是使用v-bind的方式动态的添加class来改变元素的css。

同时我还提供了三种方式来选择题目：

  • 默认顺序方式（index不断的自加，达到顺序出题的效果）
  • 随机方式（index是在一定范围内的随机数，能够达到随机的效果）
  • 指定方式（index是可以根据用户的需要所自定义，但是一定要在范围之内，要不然没有反应）

router

前者是用来指定点击后的路径的，后者是点击后展示的组件的。
