Basic concepts

Linear List: A linear table is a finite sequence of N data elements(a1,a2,…,ai-1,ai,ai+1,…,an).

each data element in the collection has only one direct precursor

each data element in the collection has only one direct successor

Characteristics of linear table

  • Identity：The linear table consists of the same kind of data elements, and each ai must belong to the same data object.ai
  • Fineness：A linear table consists of a limited number of data elements, and the table length is the number of data elements in the table.
  • Orderliness：There is order even relationship between adjacent data elements in linear table<ai,ai+1>.
