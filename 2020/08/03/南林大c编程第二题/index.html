题目

设计一个stack类的结构，该类可以完成栈的功能。该栈的元素类型为int，最大深度为100。它能完成push和pop方法，功能分别为压栈和出栈。
如下main()函数可以执行。
void main()
{
stack s1;
int i;
for(i = 1; i < 10;i++)
s1.push(i);
for(i = 1; i <6;i++)
cout << s1.pop()<<”,”;
s1.push(200);
cout << s1.pop()<<endl;
}
执行的结果为
9,8,7,6,5,200

分析

第二题也没有什么难度，主要要注意一下栈的溢出和触底的检验操作

代码

#include

#define maxN 100 // 栈的最大容量

using namespace std;

class stack{
private:
int top; // 栈顶指针，初始化的时候指向-1
int a[maxN]; // 用数组来模拟栈
public:
stack(); // 空参构造器
void push(int num); // push方法
int pop(); // pop方法
};

// 初始化栈
stack::stack() {
top = -1;
}

void stack::push(int num) {
if (top < maxN-1) { // 检验栈有没有溢出
top++;
a[top] = num;
}
}

int stack::pop() {
if (top > -1) { // 检验栈有没有触底
top–;
return a[top+1];
}
return -1;
}

int main()
{
stack s1;
int i;

for(i = 1; i < 10;i++)
    s1.push(i);
for(i = 1; i <6;i++)
    cout << s1.pop()<<",";
s1.push(200);
cout << s1.pop()<<endl;
return 0;

}
